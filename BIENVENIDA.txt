⚡ COMENZAR EN 3 PASOS (15 MINUTOS):
═══════════════════════════════════════════════════════════════════════════════

   PASO 1: INSTALAR DEPENDENCIAS (5 minutos)
   ─────────────────────────────────────────
   $ cd C:\Users\ptic252\Documents\GitHub\ACME-OPERACIONE
   $ python -m venv venv
   $ venv\Scripts\activate
   $ pip install -r requirements.txt


   PASO 2: CREAR TABLAS EN SUPABASE (5 minutos)
   ─────────────────────────────────────────────
   • Abre Supabase > SQL Editor
   • Abre archivo: setup_database.sql
   • Copia y pega TODO el contenido
   • Ejecuta (Ctrl+Enter)
   • Deberías ver: "✅ Tablas creadas exitosamente!"


   PASO 3: CONFIGURAR Y EJECUTAR (5 minutos)
   ────────────────────────────────────────
   • Edita .env con tus credenciales:
     - SUPABASE_URL (de Supabase)
     - SUPABASE_KEY (de Supabase)
     - DATABASE_URL (de Supabase)
     - CULQI_PUBLIC_KEY (de Culqi)
     - CULQI_PRIVATE_KEY (de Culqi)

   • Ejecuta:
     $ uvicorn main:app --reload --host 0.0.0.0 --port 8000

   • Abre: http://localhost:8000/docs
   • ¡Prueba los endpoints! 🎯


🎯 FLUJO COMPLETO DE COMPRA:
═══════════════════════════════════════════════════════════════════════════════

   1. Cliente VE PRODUCTOS
      └─> GET /api/products

   2. Cliente CREA PEDIDO
      └─> POST /api/orders
         → Se crea cliente automáticamente
         → Se descuenta stock

   3. Cliente GENERA TOKEN CULQI
      └─> Frontend genera token

   4. Backend PROCESA PAGO
      └─> POST /api/pay
         → Transacción exitosa/fallida
         → Pedido cambia a PAGADO

   5. Cliente VE SU PEDIDO
      └─> GET /api/orders/{id}
