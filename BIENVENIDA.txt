⚡ COMENZAR EN 3 PASOS (15 MINUTOS):
═══════════════════════════════════════════════════════════════════════════════════

   PASO 1: INSTALAR DEPENDENCIAS (5 min)
   $ python -m venv venv
   $ venv\Scripts\activate
   $ pip install -r requirements.txt

   PASO 2: CREAR TABLAS EN SUPABASE (5 min)
   • Abre Supabase > SQL Editor
   • Copia setup_database.sql
   • Ejecuta (Ctrl+Enter)
   • Deberías ver: "✅ Tablas creadas exitosamente!"

   PASO 3: CONFIGURAR Y EJECUTAR (5 min)
   • Edita .env con tus credenciales
   • Ejecuta: uvicorn main:app --reload
   • Abre: http://localhost:8000/docs
   • ¡Prueba los endpoints!
